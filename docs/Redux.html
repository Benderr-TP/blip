Blip's usage of Redux

As of the spring of 2016, we have migrated the application state management in blip to Redux. Redux is a lightweight but powerful state container for JavaScript applications that takes inspiration equally from (a) Facebook's Flux application architecture (especially its emphasis on one-way data flow) and (b) functional programming, in particular Elm, a functional programming language for building GUIs on the web.

Please read the general Redux @ Tidepool documentation before reading the documents listed below giving details of our usage of Redux in the Tidepool uploader.
